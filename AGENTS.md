# AGENTS.md - vm-store-linktree

## Project
Astro + Tailwind CSS linktree para tienda de ropa urbana en Colombia.

## Commands
```bash
npm run dev      # Dev server
npm run build   # Build producción
npm run preview # Preview build
```

## Tech
- Node >=22.12.0
- Astro 6.x
- Tailwind 4.x
